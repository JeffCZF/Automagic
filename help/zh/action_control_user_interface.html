控制用户界面

控制用户界面动作利用设备的无障碍功能来控制用户界面，可以自动按下按钮、长按元素、滚动列表以及更改复选框和单选按钮的值。

注意：此动作通过模拟用户输入来控制用户界面，因此屏幕必须处于活动状态、已解锁且用户界面必须显示在屏幕上。
要使用此动作，需要在首选项中启用 Automagic 的无障碍服务。至少需要 Android 4.1（Jelly Bean）。

注意：在某些设备上，如果屏幕亮度或屏幕滤镜应用在其他控件上显示覆盖层，则无法开启无障碍服务。当无法在系统设置中按下“确定”按钮时，请暂时禁用此类应用。

 

示例：
  • 启动位置设置页面并启用/禁用 GPS。
  • 按下“立即检查”按钮启动设备更新检查。

清除所有通知： 
notifications();//展开通知栏
sleep(1000);//等待一秒让动画完成
clicked = click("清除所有通知");//点击按钮
if(not clicked) //检查按钮是否成功点击
{
    back();//当清除按钮不可用时隐藏通知栏
}
切换 GPS 开/关：
使用动作 启动应用：设置/位置服务 直接启动 GPS 设置页面 
sleep(1000);//等待设置页面显示
click("GPS 卫星");//切换设置开/关
sleep(500);
back();//返回上一个屏幕
打开应用抽屉： 
home();//确保主屏幕正在显示
sleep(1000);//等待一秒让动画完成
click("应用");//打开应用抽屉
