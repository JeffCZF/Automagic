初始化变量媒体会话（实验性）

初始化变量媒体会话（实验性）动作将指定媒体播放器的媒体会话信息填充到变量中。

 

示例：
  • 检查 Google Play Music 是否正在播放曲目。
设置
包名
要检查的媒体播放器名称。支持通配符（*、?）。支持变量。
变量是存储值的容器，可在许多动作和条件中使用，以动态定义文本的一部分。
参见动作脚本说明。

提供的变量
playback_state
当前状态（0=无，1=停止，2=暂停，3=播放，4=快进，5=快退，6=缓冲，7=错误，8=连接中，9=跳至上一首，10=跳至下一首，11=跳至队列中的项目）
playback_position
当前位置（毫秒）
title
标题
subtitle
副标题
description
描述
icon
当前图像
icon_uri
加载图像的 URI
media_duration
曲目持续时间（毫秒）
