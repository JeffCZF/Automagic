符号 位置 本地化？ 含义
0 数字 是 数字。
@ 数字 否 有效数字。
# 数字 是 数字，不显示前导零。
. 数字 是 小数分隔符或货币小数分隔符。
- 数字 是 负号。
, 数字 是 分组分隔符。
E 数字 是 在科学记数法中分隔尾数和指数。在前缀或后缀中不需要引号。
+ 指数 是 在正指数前添加本地化的加号。在前缀或后缀中不需要引号。
; 子模式边界 是 分隔正子模式和负子模式。
% 前缀或后缀 是 乘以 100 并显示为百分比。
‰ (\u2030) 前缀或后缀 是 乘以 1000 并显示为千分比。
¤ (\u00A4) 前缀或后缀 否 货币符号，替换为货币符号。如果加倍，则替换为国际货币符号。如果出现在模式中，则使用货币小数分隔符而不是小数分隔符。
' 前缀或后缀 否 用于在前缀或后缀中引用特殊字符，例如 "'#'#" 将 123 格式化为 "#123"。要创建单引号本身，请连续使用两个："# o''clock"。
* 前缀或后缀边界 是 填充转义，位于填充字符之前。
