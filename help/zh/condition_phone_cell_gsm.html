电话蜂窝 GSM

条件 电话蜂窝 GSM 检查手机是否连接到了指定的目标小区。

 

示例：
  • 检测到家中手机小区时打开 WiFi。
设置
方法
应使用传统方法、现代方法还是两者来查找小区。哪种方法效果最好高度依赖于设备。
位置区码 (LAC) 列表
位置区码的逗号分隔列表。一个位置区是一组基站，通常覆盖一个较大的区域。支持全局模式 (*, ?)。支持变量。
小区 ID (CID) 列表
小区 ID 的逗号分隔列表。支持全局模式 (*, ?)。支持变量。
变量是一个值的容器，可在许多动作和条件中用于动态定义文本的一部分。
有关说明，请参阅动作脚本。

提供的变量
lac
当前连接的位置区码 (LAC)，未知时为 -1
cid
当前连接的小区 ID (CID)，未知时为 -1
psc
主同步码 (PSC)，未知时为 -1
neighboring_cells
邻近小区列表。不可用时为 null 或空
cells (Android 4.2+)
系统已知的所有小区的列表。不可用时为 null 或空
