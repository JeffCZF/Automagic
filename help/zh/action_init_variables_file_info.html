初始化变量文件信息

初始化变量文件信息动作将文件的相关信息作为变量提供给流程。

 

示例：
  • 检查文件是否超过一周，并删除该文件。
设置
文件
要检查的文件。支持变量。
文件类型
用于存储文件类型的变量（dir、file 或 other）。
文件是否存在
用于存储文件是否存在的变量。
文件大小
用于存储文件大小（字节）的变量。
目录递归
对于目录，是收集所有直接包含的文件的大小，还是递归遍历并汇总所有子目录的大小。
文件最后修改时间
用于存储最后修改时间戳的变量。
变量是存储值的容器，可在许多动作和条件中使用，以动态定义文本的一部分。
参见动作脚本说明。

提供的变量
file_type
文件类型（dir、file 或 other）
file_exists
文件是否存在（true 或 false）
file_size
文件大小（字节）（或目录中文件的总和）
file_last_modified
最后修改时间戳
