初始化变量联系人信息

初始化变量联系人信息动作加载指定联系人的信息，并将信息存储在一个映射中。

提示： 可以使用条件 调试对话框 来显示映射中存储的键和值。

 

示例：
  • 显示一个联系人的所有电话号码。

联系人的JSON示例：

        {
          "display_name": "Contact XYZ", // 联系人姓名
          "photo_thumb_uri": "content://com.android.contacts/contacts/124/photo", // 小图片的URI
          "photo_uri": "content://com.android.contacts/display_photo/3", // 大图片的URI
          "last_time_contacted": 1479667934531, // 上次联系的时间点，0表示未知
          "starred": true, // 联系人是否在收藏夹中
          "times_contacted": 1, // 联系次数（根据联系人应用）
          "phone_list": [{ // 电话号码列表
            "number": "+41987654321",
            "type": 2,
            "type_label": "Mobile"
          }, ... ],
          "organization_list": [{ // 组织列表
            "company": "Company X",
            "type": 2,
            "type_label": "Other",
            "title": "Company"
          }, ... ],
          "note_list": [{ // 备注列表
              "note": "Note 1"
            }, ... ],
          "nickname_list": [{ // 昵称列表
            "nickname": "MR. incredible",
            "type": 0,
            "type_label": "Other"
          }, ... ],
          "email_list": [{ // 电子邮件地址列表
            "email": "example@gmail.com",
            "type": 1,
            "type_label": "Home"
          }, ... ],
          "relation_list": [{ // 关系列表
            "relation": "Person 123",

请注意，未来的Automagic版本可能会添加额外的键值对。


在动作脚本中访问值的示例：

        name = contact_info["display_name"];
        photo_uri = contact_info["photo_uri"];
        phone1 = contact_info["phone_list"][0]["number"];
        phone2 = contact_info["phone_list"][1]["number"];
        phone2type = contact_info["phone_list"][1]["type_label"];
        nickname1 = contact_info["nickname_list"][0]["nickname"];
设置
搜索类型
定义如何搜索联系人。使用找到的第一个联系人来提供信息。
搜索值
用于搜索联系人的值。支持变量。
变量是存储值的容器，可在许多动作和条件中使用，以动态定义文本的一部分。
参见动作脚本说明。

提供的变量
contact_info
包含查询信息的映射
