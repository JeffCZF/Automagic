生物识别身份验证对话框

条件 生物识别身份验证对话框 显示系统的身份验证对话框，并验证生物特征（如指纹）。

当指纹扫描失败几次后，系统暂时不会显示该对话框。

 

示例：
  • 在删除文件之前出于安全原因显示身份验证对话框。
设置
标题
要显示的标题（必需）。支持变量。
副标题
要显示的副标题（可选）。支持变量。
描述
要显示的描述（可选）。支持变量。
取消按钮标签
取消按钮的标签（必需）。支持变量。
激活超时
当用户在指定时间内没有响应时，是否应使用自动超时。
持续时间
自动关闭对话框并继续流程的时间量。
超时时的值
流程应继续到 true 分支还是 false 分支。
变量是一个值的容器，可在许多动作和条件中用于动态定义文本的一部分。
有关说明，请参阅动作脚本。

提供的变量
operation
用于关闭对话框的操作。

可以是以下之一：
  • success：用户成功验证时
  • failure：用户验证失败时
  • timeout：对话框因超时关闭时
  • cancel：用户按下返回按钮或返回键时
error_code
对话框因错误关闭时的错误代码
error_message
对话框因错误关闭时的错误描述文本
