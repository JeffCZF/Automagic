初始化变量图像文件

初始化变量图像文件动作将图像的像素数据加载到指定的局部变量中，以便使用动作 修改图像 修改图像或使用脚本函数 getPixelColor 读取像素的颜色值。

警告： 加载大图像会占用大量内存，在最坏的情况下可能导致 Automagic 崩溃。建议仅将一张图像读入内存，并且不要加载非常大的图像。

 

示例：
  • 加载图像，缩小尺寸，并通过邮件发送较小的文件。
设置
文件
图像文件的路径或 URL。支持变量。
变量
用于存储像素数据的局部变量名称。支持变量。
变量是存储值的容器，可在许多动作和条件中使用，以动态定义文本的一部分。
参见动作脚本说明。

提供的变量
image_data
图像的局部变量
image_width
图像的宽度
image_height
图像的高度
