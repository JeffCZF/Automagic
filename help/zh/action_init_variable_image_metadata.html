初始化变量图像元数据

初始化变量图像元数据动作从图像中提取元数据（例如 Exif 标签），并将值存储在一个映射中（键=标签的名称/标签，值=标签的描述或原始值）。

在动作 脚本 中读取 GPS 位置的示例：
location = getMapValue(image_metadata, "gps_location");

提示： 可以使用条件 调试对话框 显示映射中存储的键和值。

 

示例：
  • 从 JPEG 图像中提取 GPS 位置和缩略图。
设置
文件
图像文件的路径。支持变量。
变量
用于存储包含值的映射的变量名称。支持变量。
解析日期
定义是否应将日期字符串转换为适当的日期/时间值，还是将字符串作为字符串存储在映射中。
包含元数据目录名称
是否在每个读取标签的键中包含元数据目录的名称。
获取原始值
定义是否应将元数据值转换为人类可读的形式，还是按原样加载值。
提取缩略图
定义是否应从元数据中提取可用的缩略图并存储在变量 thumbnail_image 中。
变量是存储值的容器，可在许多动作和条件中使用，以动态定义文本的一部分。
参见动作脚本说明。

提供的变量
image_metadata
包含提取的标签名称/标签以及每个标签的提取值的映射
thumbnail_image
从图像元数据中提取的缩略图图像
