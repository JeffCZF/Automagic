插件（实验性）

触发器 插件（实验性） 在插件的条件更改为 true 和/或 false 时执行流程。
警告：此功能是实验性的，可能不适用于 Google Play 中所有现有的插件。我们无法为插件提供支持。

注意：可以使用来自 Locale 应用的接口描述来定义插件：Locale 开发者文档 和 Tasker：Tasker 插件扩展。

设置
插件
在设备上找到的所有可用插件的列表。
条件
  • 变为 true：当插件中测量的条件从 false 或 undefined 更改为 true 时
  • 变为 false：当插件中测量的条件从 true 或 undefined 更改为 false 时
变量是一个值的容器，可在许多动作和条件中用于动态定义文本的一部分。
有关说明，请参阅动作脚本。

提供的变量
triggertime
触发器执行的时间
condition
条件变为 true 还是 false
