应用进程正在运行

条件 应用进程正在运行 检查托管应用的进程当前是否正在运行。

注意：即使进程可能不再需要且有足够的空闲内存可用，Android 也会将进程保留在内存中。

注意：Google 已在 Android 5.1.1 上移除了用于访问此信息的官方 API，这意味着该条件现在被标记为 非官方支持。
注意：在大多数运行 Android 7 的设备上，用于访问此信息的非官方变通方法已不再有效。

设置
包名
要检查的进程名称。支持变量。
变量是一个值的容器，可在许多动作和条件中用于动态定义文本的一部分。
有关说明，请参阅动作脚本。

提供的变量
pid
进程的 ID
importance (Android <5.1.1)
进程的重要性。值越小，重要性越高。
lru (Android <5.1.1)
最近最少使用。值越小表示最近使用过。
process_state（并非在所有 Android 版本上都可用，Android <5.1.1）
进程状态（2=托管当前顶层活动的进程）
uid
进程的用户 ID
