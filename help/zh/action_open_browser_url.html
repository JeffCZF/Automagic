在浏览器中打开 URL

在浏览器中打开 URL动作使用指定的 URL 打开浏览器。

提示： 浏览器也可用于下载文件，只需使用指向 PDF 或其他可下载文件的 URL。

 

示例：
  • 每天早上打开新闻页面。
  • 每天早上 5 点下载 PDF。
设置
URL
要打开的 URL。支持变量。

提示： 您可以使用变量来打开随日期动态变化的 URL。

http://test.com/newspaper_{triggertime,dateformat,yyyyMMdd}.pdf

将被替换为

http://test.com/newspaper_20120123.pdf
使用指定应用
是使用定义的应用打开 URL，还是让系统选择最合适的应用。
包名
用于打开 URL 的应用。支持变量。
类名
应用内用于打开 URL 的组件。支持变量。
