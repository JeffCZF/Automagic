初始化变量包信息

初始化变量包信息动作加载指定应用/包的信息，并将信息存储在一个映射中。

提示： 可以使用条件 调试对话框 来显示映射中存储的键和值。

 

示例：
  • 加载应用使用的权限列表，并检查哪些权限已被授予。

格式

映射包含包名作为键，以及包含查询信息的映射作为值。JSON 示例：

        {
          "package1": {"version_name": xyz, "version_code": 123},
          "package2": {"version_name": xyz, "version_code": 123}
        }

请注意，未来的 Automagic 版本可能会添加额外的键值对。

设置
类型
查询所有已安装包的信息还是仅查询少数选定的包。
包名
要查询的包列表。支持通配符（*、?）。支持变量。
包含权限信息
是否查询应用请求的权限信息。
包含活动信息
是否查询应用定义的活动信息。
包含接收器信息
是否查询应用定义的接收器信息。
包含提供者信息
是否查询应用定义的内容提供者信息。
包含服务信息
是否查询应用定义的服务信息。
变量
用于存储结果的变量名称。
变量是存储值的容器，可在许多动作和条件中使用，以动态定义文本的一部分。
参见动作脚本说明。

提供的变量
package_info
包含查询信息的映射
