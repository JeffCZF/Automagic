输入对话框

输入对话框动作显示一个输入对话框，包含输入文本字段、密码字段、单选或多选列表。

 

示例：
  • 询问用户是否要关闭 WiFi。
  • 询问用于 FTP 上传的密码。
  • 询问晚上是否应激活飞行模式。如果用户在 30 秒内没有响应，则激活飞行模式。
设置
输入对话框类型
要显示的输入对话框类型。

可以是以下之一：
  • 多行文本：用于多行纯文本输入
  • 单行文本：用于单行纯文本输入
  • 密码：用于密码输入
  • PIN：用于 PIN 输入
  • 数字：用于输入数字字符
  • 电话号码：用于输入电话号码中使用的字符
  • 日期：显示日期选择器
  • 时间：显示时间选择器
  • 日期和时间：显示带有日期和时间选择器的对话框
  • 单选菜单：显示值列表，类似于没有“确定”、“取消”按钮的菜单
  • 多选：显示值列表，用户可以选择多个值
标题
输入对话框的标题。支持变量。
提示（仅适用于 文本 和 密码）
在输入对话框中显示的问题或输入描述。支持变量。
列表值（仅适用于 单选 和 多选）
以逗号分隔的值列表，作为用户可供选择的值显示。支持变量。
默认值
用户未更改输入值时提供的默认值。当用户取消对话框或按返回键时，也会提供此值。支持变量。
用户是否取消了对话框可以使用变量 operation 来判断（见下文）。

日期/时间输入类型的默认值必须按照以下指定格式进行格式化：
  • 日期：模式 yyyy-MM-dd，例如 2014-01-25 或使用变量 {variable,dateformat,yyyy-MM-dd}
  • 时间：模式 HH:mm（24小时制），例如 14:20 或使用变量 {variable,dateformat,HH:mm}
  • 日期和时间：模式 yyyy-MM-dd HH:mm（时间使用24小时制），例如 2014-01-25 14:20 或使用变量 {variable,dateformat,yyyy-MM-dd HH:mm}
未根据上述规则格式化的值将被忽略。
字体大小
用于输入文本值的字体大小。
激活超时
是否应在用户未在指定时间内反应时，自动超时并向流程提供值。
持续时间
自动关闭对话框并继续流程的时间量。
超时时的值
发生超时时提供给流程的值。支持变量。
选项
  • 锁定时显示：即使设备已锁定也显示输入对话框
  • 开启屏幕：首次显示对话框时开启屏幕
  • 保持屏幕开启：只要对话框显示，就保持屏幕开启（电源按钮仍允许关闭屏幕）
  • 放大对话框宽度：放大对话框宽度以覆盖几乎整个屏幕
  • 键盘提示大写句子：向键盘提示句子应以大写字母开头
变量是存储值的容器，可在许多动作和条件中使用，以动态定义文本的一部分。
参见动作脚本说明。

提供的变量
operation
用于关闭对话框的操作。

可以是以下之一：
  • ok：用户按下“确定”按钮
  • cancel：用户按下“取消”按钮或返回键
  • timeout：对话框因超时关闭
value
对话框中选择的值，或者对于 多选 类型是值的列表，或者对于日期/时间类型的输入对话框是日期。 对于类型 时间，value 是当前日期下输入的时间。
index
单选 列表中所选值的索引，如果无法定义则为 -1
indices
多选 列表中所选值的索引列表，对于每个无法定义的值则为 -1
