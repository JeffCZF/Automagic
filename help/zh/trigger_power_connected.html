电源

触发器 电源 在电源连接到设备（充电器或 USB 数据线）时执行。

 

示例：
  • 连接电源时打开 WiFi。
  • 连接电源时增加屏幕亮度。
  • 断开电源时将屏幕亮度设置为自动。
设置
电源已连接/已断开
已连接
选中时，在电源连接时执行流程。
  • 交流适配器：使用交流适配器充电时
  • USB：通过 USB 连接充电时
  • 无线：无线充电时

已断开
选中时，在电源断开时执行流程。

变量是一个值的容器，可在许多动作和条件中用于动态定义文本的一部分。
有关说明，请参阅动作脚本。

提供的变量
triggertime
触发器执行的时间
battery_level
电池的电量级别
battery_percentage
电量百分比
battery_plugged
电池是否已插入电源（1=交流适配器，2=USB，4=无线）或未插入（0=未插入）
battery_present
电池是否存在
battery_scale
最大电池电量级别
battery_status
电池的状态（1=未知，2=充电中，3=放电中，4=未充电，5=已满）
battery_technology
电池的技术
battery_temperature
电池的温度（以十分之一摄氏度为单位）
battery_voltage
当前的电池电压
battery_health
电池的健康状况
battery_charge_counter（Android 5+）
电池容量（微安时）。不可用时为 null
battery_current_now（Android 5+）
瞬时电池电流（微安）（正值=充电，负值=放电）。不可用时为 null
battery_current_average（Android 5+）
平均电池电流（微安）（正值=充电，负值=放电）。不可用时为 null
battery_energy_counter（Android 5+）
电池剩余能量（纳瓦时）。不可用时为 null
