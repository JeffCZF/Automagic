设置数据漫游状态

动作 设置数据漫游状态 打开或关闭全局数据漫游设置。

警告： 此功能需要 root 权限，不受 Android 官方支持，可能无法在所有设备上正常工作。
注意： 在某些 Android 版本上，可以使用 adb 授予执行动作的权限而无需 root：

  1. 在计算机上安装 adb（搜索互联网了解如何安装 adb）
  2. 在 Android 设备的 系统设置 -> 开发者选项 中启用 USB 调试
  3. 将 Android 设备连接到计算机
  4. 在终端中执行以下命令：adb shell pm grant ch.gridvision.ppam.androidautomagic android.permission.WRITE_SECURE_SETTINGS

示例：
  • 每天早晨在国外激活数据漫游 10 分钟以同步邮件，然后在一天中的其余时间再次停用数据漫游。
