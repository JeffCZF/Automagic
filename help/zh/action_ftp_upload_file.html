FTP 上传文件

FTP 上传文件动作将单个文件上传到 FTP 服务器。

 

示例：
  • 将 Gleeo Time Tracker 中的所有数据备份上传到您公司的服务器。
设置
协议
使用常规 FTP（未加密）、FTPS（基于 SSL/TLS 的隐式 FTP，默认端口 990）还是 FTPES（基于 SSL/TLS 的显式 FTP）。
数据通道保护（仅适用于 FTPS 和 FTPES）
数据通道上传输的数据是否应加密（P）或不加密（C）。
服务器/端口
FTP 服务器的名称或 IP 及端口（默认 21）。支持变量。
用户名
连接的用户名。支持变量。
密码
连接的密码。支持变量。
本地文件
要上传的本地文件的绝对路径。支持变量。
示例：
  • /mnt/sdcard/export_{triggertime,dateformat,yyyy_MM_dd}.csv 变为 /mnt/sdcard/export_2011_02_13.csv
服务器上的路径（包含文件名）
文件在 FTP 服务器上的绝对路径。路径必须包含目标文件的文件名。这允许您在服务器上使用不同的名称存储文件。支持变量。
控制编码
是否对控制通道使用特殊编码（文件名的编码，而不是文本文件内容的编码）。支持变量。
显示进度通知
是否在状态栏上显示指示操作进度的通知。
