GPS 状态事件

触发器 GPS 状态事件 在系统开始或停止使用 GPS 时执行。

 

示例：
  • 当系统开始使用 GPS 以获取信息时振动。
设置
GPS 状态事件
事件的类型：
  • 已开始：当系统开始使用 GPS 时执行触发器
  • 已停止：当系统停止使用 GPS 时执行触发器
  • 收到首次定位：当系统使用 GPS 且刚刚确定首次定位时执行触发器
  • 卫星状态已更改：当卫星状态更改时执行触发器（通常非常频繁）
变量是一个值的容器，可在许多动作和条件中用于动态定义文本的一部分。
有关说明，请参阅动作脚本。

提供的变量
triggertime
触发器执行的时间
gps_event_type
事件类型（1=已开始，2=已停止，3=首次定位，4=卫星状态）
satellites
包含字符串的列表，每个字符串包含一颗卫星的信息
time_to_first_fix
从打开 GPS 到收到首次定位之间的毫秒数，当尚未确定定位时包含 0 或上一个值
