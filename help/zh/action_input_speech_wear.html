语音输入（Android Wear）

语音输入（Android Wear）动作启动设备的语音识别，并将识别出的文本通过变量 value 提供给流程。语音识别引擎通常需要网络访问。

 

示例：
  • 使用语音命令控制 Automagic。
设置
Android Wear 设备
目标设备。支持变量。
UI 模式
是显示用户界面，还是在后台执行语音识别。
提示
在用户界面中显示的可选标题。此设置可能被引擎忽略。支持变量。
语言
可选，语音识别引擎中使用的期望语言。此设置可能被引擎忽略。支持变量。
变量是存储值的容器，可在许多动作和条件中使用，以动态定义文本的一部分。
参见动作脚本说明。

提供的变量
value
识别出的文本
operation
识别是否成功（ok）或失败（cancel）
