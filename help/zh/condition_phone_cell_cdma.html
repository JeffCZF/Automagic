电话蜂窝 CDMA

条件 电话蜂窝 CDMA 检查手机是否连接到了指定的目标小区。

 

示例：
  • 检测到家中手机小区时打开 WiFi。
设置
方法
应使用传统方法、现代方法还是两者来查找小区。哪种方法效果最好高度依赖于设备。
小区列表
小区 ID 的逗号分隔列表。支持全局模式 (*, ?)。支持变量。
变量是一个值的容器，可在许多动作和条件中用于动态定义文本的一部分。
有关说明，请参阅动作脚本。

提供的变量
triggertime
触发器执行的时间
cell_id
小区 ID，未知时为 null
base_station_id
基站 ID，未知时为 -1
network_id
网络 ID，未知时为 -1
system_id
系统 ID，未知时为 -1
base_station_latitude
基站的纬度，单位为四分之一度秒，未知时为 2147483647。将值除以 14400 可获得十进制度数。并非所有网络都提供有意义的值。
base_station_longitude
基站的经度，单位为四分之一度秒，未知时为 2147483647。将值除以 14400 可获得十进制度数。并非所有网络都提供有意义的值。
neighboring_cells
邻近小区列表。不可用时为 null 或空
cells (Android 4.2+)
系统已知的所有小区的列表。不可用时为 null 或空
