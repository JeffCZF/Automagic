下载 URL

动作 下载 URL 直接在后台将文件下载到指定目录。

 

示例：
  • 每天早上下载 PDF 报纸。
设置
URL
要下载的 URL。支持变量。
示例：
  • http://example.com/newspaper_{triggertime,dateformat,yyyyMMdd}.pdf 变为 http://example.com/newspaper_20110213.pdf
目录
放置下载文件的目录。支持变量。
文件名
下载文件的文件名。如果未提供，则使用 URL 确定文件名。支持变量。
显示进度通知
是否在状态栏上显示指示操作进度的通知。
变量是存储值的容器，可在许多动作和条件中使用，以动态定义文本的一部分。
参见动作脚本说明。

提供的变量
file_path
下载文件的目标路径
