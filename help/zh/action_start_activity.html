启动活动

动作 启动活动 使用指定的 Intent 启动一个活动。另请参阅 Android Intent 类的文档。
提示：某些活动需要特殊权限才能启动。当活动启动失败时，请在测试动作后检查日志。

 

示例：
  • 启动其他应用程序公开的特定操作。
  • 发起 Skype 通话：
    • 动作：android.intent.action.VIEW
    • 数据 URI：skype:echo123
  • 拨号（不呼出）：
    • 动作：android.intent.action.DIAL
    • 数据 URI：tel:99
  • 启动带有预定义主题的邮件应用：
    • 动作：android.intent.action.SEND
    • 数据 MIME 类型：text/plain
    • 附加数据：putString("android.intent.extra.SUBJECT", "test")
设置
动作
要在 Intent 中使用的动作。支持变量。
类别列表
要添加到 Intent 中的类别列表，以逗号分隔。支持变量。
数据 URI
要在 Intent 上设置的数据 URI。支持变量。
Mime 类型
要在 Intent 上设置的 MIME 类型。支持变量。
包名
要启动的特定包。支持变量。
类名
要启动的特定活动。支持变量。
启动边界 (Android 7+)
是否应在定义的位置和大小启动活动。此功能仅在支持自由形式模式的设备上有效。 支持变量。
标志列表
要添加到 Intent 的标志。请注意，标志 Intent.FLAG_ACTIVITY_NEW_TASK 始终会添加到 Intent 中。
附加数据
用于向 Intent 添加无限数量附加数据的脚本。

除了 Automagic 的常规内置函数外，还可以使用以下函数向 Intent 添加附加数据：
  • putString(key, value)
  • putBoolean(key, value)
  • putFloat(key, value)
  • putDouble(key, value)
  • putByte(key, value)
  • putShort(key, value)
  • putChar(key, value)
  • putInt(key, value)
  • putLong(key, value)
  • putUri(key, value)
  • putBooleanArray(key, value)
  • putByteArray(key, value)
  • putIntArray(key, value)
  • putIntegerArrayList(key, value)
  • putLongArray(key, value)
  • putFloatArray(key, value)
  • putDoubleArray(key, value)
  • putShortArray(key, value)
  • putCharArray(key, value)
  • putStringArray(key, value)
  • putStringArrayList(key, value)
  • putUriArray(key, value)
  • putUriArrayList(key, value)

示例： 
putString("key1", "Hello");
putBoolean("key2", true);
putDouble("key3", 1.23);
putInt("key4", 42);
putInt("key5", a+4*b);
有关脚本语言的说明，请参阅动作脚本。

变量是一个值的容器，可在许多动作和条件中用于动态定义文本的一部分。
有关说明，请参阅动作脚本。

提供的变量
result_code （仅当启用了 等待结果 时）
返回的结果代码
operation （仅当启用了 等待结果 时）
字符串 "ok"（当 result_code 为 -1 时），否则为 "cancel"
action （仅当启用了 等待结果 时）
接收到的 Intent 的动作
data_string （仅当启用了 等待结果 时）
接收到的 Intent 的数据（字符串形式）
categories （仅当启用了 等待结果 时）
类别列表
scheme （仅当启用了 等待结果 时）
接收到的 Intent 的协议
type （仅当启用了 等待结果 时）
接收到的 Intent 的显式设置的 MIME 类型
