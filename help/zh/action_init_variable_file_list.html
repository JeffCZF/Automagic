初始化变量文件列表

初始化变量文件列表动作列出目录中的所有文件或仅新添加的文件，并将列表存储在变量中。

 

示例：
  • 将目录中的所有文件通过邮件发送到您的邮箱。
  • 将 Gleeo Time Tracker 的所有新备份文件发送到您的收件箱。
设置
变量
用于保存文件列表的变量名称。
文件模式
要列出的文件。文件模式可以包含通配符，如 * 和 ?。

示例：
  • /mnt/sdcard/folder/*
  • /mnt/sdcard/GTT-export-20120101.csv
  • /mnt/sdcard/GTT-export-*.csv
  • /mnt/sdcard/file1,/mnt/sdcard/file2,/mnt/sdcard/file3
仅包含更改
是存储所有匹配 文件模式 中指定的模式的文件，还是仅存储新文件。
如果文件在上次执行此操作时不存在，或者文件大小发生变化，或者自上次执行此操作以来文件的修改时间发生变化，则认为该文件是新文件。
包含文件属性
选中时，为每个文件返回包含路径、文件类型、文件大小和文件日期的属性列表。
变量是存储值的容器，可在许多动作和条件中使用，以动态定义文本的一部分。
参见动作脚本说明。

提供的变量
files
文件列表
