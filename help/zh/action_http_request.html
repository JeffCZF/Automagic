HTTP 请求

HTTP 请求动点使用指定的 HTTP 请求方法从服务器请求资源。

注意： 此操作用于技术性请求（REST 等）并使用脚本处理响应。如果您只是想下载文件，请使用动作 下载 URL。
该操作在出现一般错误时以状态码 -1 终止。当甚至无法建立与服务器的连接时（网络不可用、无法解析 IP 等），它将因错误而终止。使用异常连接来处理此类错误。

 

示例：
  • 获取天气服务的网页并测试是否包含 rain 这个词。
  • 从在线零售商的 REST 服务获取产品价格。
设置
URL
要请求的 URL。支持变量。
示例：
  • http://example.com/weather.html?ts={triggertime,dateformat,yyyyMMdd} 变为 http://example.com/weather.html?ts=20110213
验证证书（https）
是否应检查证书。对于忽略自签名证书的错误可能很有用。
认证
是否应使用基本认证将用户名和密码传递给服务器。
用户名
要传递给服务器的用户名。支持变量。
密码
要传递给服务器的密码。支持变量。
客户端证书
是否应使用客户端证书。
客户端证书路径
PKCS#12 文件的路径。支持变量。
客户端证书密码
访问客户端证书的密码。支持变量。
网络类型
使用当前可用的默认网络，还是在 WiFi 或蜂窝网络上发送请求。
请求方法
要使用的 HTTP 请求方法。并非所有设备都支持所有方法。
内容类型
请求中用作标头 Content-Type 的内容类型。支持变量。
表单字段列表
以逗号分隔的 key=value 列表，作为 application/x-www-form-urlencoded 发送到服务器。值会自动进行 URL 编码。支持变量。

示例：Firstname=Hugo,Lastname=Habicht
数据
要发送到服务器的文本数据。支持变量。
表单字段名称
用于上传文件的表单字段名称。支持变量。
要上传的文件
要上传的文件的路径。请求的文件名会根据指定的文件自动确定。支持变量。
超时
要使用的超时时间。
自定义 HTTP 标头
包含要发送的自定义 HTTP 标头的文本。支持变量。

每行应以标头名称和冒号开头，后跟要发送的值。

示例：
  • User-Agent: my own user agent
  • SOAPAction: "test"
将响应存储在变量或文件中
将响应存储在变量还是文件中。
变量
用于存储响应的变量名称。响应将被转换为字符串。支持变量。

警告： 大于 1MB 的响应不支持存储在变量中。
编码
用于从服务器读取响应的编码。留空以使用服务器响应指定的编码。
文件
存储文件的路径。支持变量。
变量是存储值的容器，可在许多动作和条件中使用，以动态定义文本的一部分。
参见动作脚本说明。

提供的变量
在“变量”字段中声明的名称
作为字符串的响应
status_code
数字 HTTP 状态码，发生一般错误时为 -1
error_message
当请求无法执行且 status_code 设置为 -1 时的错误消息
header_*
响应标头的所有值。变量名称使用前缀 header_ 构建，并将标头名称中的短划线替换为下划线，并将名称转换为小写。
示例：
  • 标头 Accept 存储在变量 header_accept 中
  • 标头 Accept-Encoding 存储在变量 header_accept_encoding 中
在响应中多次指定的标头值将作为列表提供。
导致无效变量名称的标头将被忽略。
