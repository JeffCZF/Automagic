Dropbox 初始化变量文件列表

Dropbox 初始化变量文件列表动作列出目录中与指定名称模式匹配的文件。

 

示例：
  • 获取文件列表，并仅下载小于 1MB 的文件。
设置
Dropbox 访问
指示当前是否可以访问 Dropbox，并允许登录您的 Dropbox 帐户。

通常不需要安装 Dropbox 应用，但可以帮助解决某些设备上的身份验证问题。
变量
用于保存文件列表的变量名称。支持变量。
文件模式
要列出的文件。文件模式可以包含通配符，如 * 和 ?。支持变量。
包含文件属性
选中时，为每个文件返回包含路径、文件类型、文件大小和文件日期的属性列表。
变量是存储值的容器，可在许多动作和条件中使用，以动态定义文本的一部分。
参见动作脚本说明。
