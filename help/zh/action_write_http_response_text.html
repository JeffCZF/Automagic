写入 HTTP 响应文本（实验性）

动作 写入 HTTP 响应文本（实验性） 将指定的文本作为对 HTTP 请求的响应写入。该流程必须由触发器 HTTP 请求 调用才能写入响应。

 

示例：
  • 当请求特定页面时，返回一个包含设备信息的页面。
设置
文本
要写入的文本。支持变量。
内容类型
内容的内容类型（例如 text/plain 或 text/html）。仅当尚未向客户端返回任何数据时，才考虑内容类型。支持变量。
自定义 HTTP 头
包含要发送的自定义 HTTP 头的文本。支持变量。

每行应以头名称和冒号开头，后跟要发送的值。

示例：
  • User-Agent: my own user agent
  • SOAPAction: "test"
完成 HTTP 响应
是否通过立即将文本发送到客户端来完成 HTTP 响应。
