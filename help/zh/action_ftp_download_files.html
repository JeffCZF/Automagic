FTP 下载文件

FTP 下载文件动作下载存储在 FTP 服务器上的文件和文件夹。

 

示例：
  • 每天早上递归下载一个包含音乐的目录。
设置
协议
使用常规 FTP（未加密）、FTPS（基于 SSL/TLS 的隐式 FTP，默认端口 990）还是 FTPES（基于 SSL/TLS 的显式 FTP）。
数据通道保护（仅适用于 FTPS 和 FTPES）
数据通道上传输的数据是否应加密（P）或不加密（C）。
服务器/端口
FTP 服务器的名称或 IP 及端口（默认 21）。支持变量。
用户名
连接的用户名。支持变量。
密码
连接的密码。支持变量。
服务器上的路径
要从服务器下载的文件或目录的逗号分隔列表。可以使用通配符模式匹配多个文件。支持变量。
示例：
  • /pub/export* 匹配 /pub 文件夹中所有以 export 开头的文件
本地目录
要将文件下载到的目标目录的绝对路径。支持变量。
控制编码
是否对控制通道使用特殊编码（文件名的编码，而不是文本文件内容的编码）。支持变量。
显示进度通知
是否在状态栏上显示指示操作进度的通知。
