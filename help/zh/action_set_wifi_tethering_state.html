设置 WiFi 网络共享状态

动作 设置 WiFi 网络共享状态 打开或关闭 WiFi 网络共享。

注意：此动作本身不提供 WiFi 网络共享功能，它仅打开该设置。部分运营商移除了 WiFi 网络共享功能。在此类手机上，此动作 无法 开启网络共享。

警告：此功能并非 Android 官方支持，可能无法在所有设备上运行。

 

示例：
  • 连接电源适配器时打开 WiFi 网络共享。
  • 当电量低于 50% 时关闭 WiFi 网络共享。
