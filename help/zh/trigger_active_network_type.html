活动网络类型

触发器 活动网络类型 在活动网络类型更改为另一种类型时执行流程。

设置
网络类型更改为
定义触发器应为哪些网络类型执行
变量是一个值的容器，可在许多动作和条件中用于动态定义文本的一部分。
有关说明，请参阅动作脚本。

提供的变量
triggertime
触发器执行的时间
network_type
网络类型（数字）（无=-1，移动=0，WiFi=1，WiMAX=6，蓝牙=7，以太网=9）
network_type_name
网络名称（字符串），如 WIFI 或 MOBILE
network_subtype
网络子类型（数字）
network_subtype_name
网络子类型名称（数字）
network_state
网络状态（CONNECTING, CONNECTED, SUSPENDED, DISCONNECTING, DISCONNECTED, UNKNOWN）
network_detailed_state
详细网络状态（IDLE, SCANNING, CONNECTING, AUTHENTICATING, OBTAINING_IPADDR, CONNECTED, SUSPENDED, DISCONNECTING, DISCONNECTED, FAILED, BLOCKED, VERIFYING_POOR_LINK, CAPTIVE_PORTAL_CHECK）
network_extra_info
附加信息（字符串，如 SSID），当可用时
network_reason
失败原因，当可用时
