设置自动同步状态

动作 设置自动同步状态 打开或关闭自动同步。

 

示例：
  • 晚上关闭自动同步。
  • 周末关闭公司帐户的自动同步。
设置
自动同步
是打开还是关闭自动同步。
全局或特定
是全局更改自动同步，还是仅针对指定的帐户/权限进行更改。更改全局开关不会更改单个帐户/权限的设置。
帐户
要更改的帐户，格式为 类型:名称。 可以使用通配符模式匹配多个帐户。支持变量。
  • * 更改所有帐户
  • Google:* 更改所有 Google 帐户
权限
要更改的帐户部分/数据库。指定 * 可更改帐户的所有部分。 可以使用通配符模式匹配多个权限。支持变量。
  • * 更改指定帐户的所有权限
  • com.android.calendar 更改指定帐户的所有日历
