设置 LineageOS 系统设置

动作 设置 LineageOS 系统设置 将指定值存储到系统的设置数据库中。

警告： 将系统设置修改为不支持的值可能会损害您的设备或导致系统崩溃。使用风险自负！

注意： 某些设置的更改不会立即被系统采纳，而是需要额外的事件发生，例如关闭并重新打开屏幕。某些设置需要通过特殊的 API 更改，在设置数据库中更改此类设置的值会被系统忽略。

注意： 在某些 Android 版本上，可以使用 adb 授予修改 安全 和 全局 类别设置的权限而无需 root：

  1. 在计算机上安装 adb（搜索互联网了解如何安装 adb）
  2. 在 Android 设备的 系统设置 -> 开发者选项 中启用 USB 调试
  3. 将 Android 设备连接到计算机
  4. 在终端中执行以下命令：adb shell pm grant ch.gridvision.ppam.androidautomagic lineageos.permission.WRITE_SECURE_SETTINGS

 

示例：
  • 更改 Automagic 中常规操作不可用的 ROM 特定设置的值。
设置
类别
  • 系统：用于常规设置
  • 安全：用于通常无法修改的设置
  • 全局：用于适用于设备所有用户的全局设置
名称
要修改的设置名称。支持变量。
值
要存储到系统设置数据库中的值。支持变量。
