设置流程状态

动作 设置流程状态 启用或禁用一个或多个流程。

 

示例：
  • 当您处于漫游状态时，禁用所有使用网络功能的流程。
  • 周六和周日禁用所有时间跟踪流程。
设置
流程模式列表
要启用或禁用的流程名称的逗号分隔列表。可以使用通配符 * 和 ? 匹配名称相似的流程。支持变量。

示例：
  • Test Flow
  • Test Flow, SMS Flow
  • *time tracking*
排除流程模式列表
即使 流程模式列表 字段包含该流程，也不影响的流程名称的逗号分隔列表。可以使用通配符 * 和 ? 匹配名称相似的流程。支持变量。
启用或禁用流程
是启用还是禁用流程。
