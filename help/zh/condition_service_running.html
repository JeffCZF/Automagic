服务正在运行

条件 服务正在运行 检查指定的服务当前是否正在运行。

设置
服务类
要检查的服务的类名。支持变量。
变量是一个值的容器，可在许多动作和条件中用于动态定义文本的一部分。
有关说明，请参阅动作脚本。

提供的变量
package_name
服务的包名
service_name
服务的显示名称
pid
此服务在其中运行的进程
uid
拥有此服务的用户 ID
active_since
此服务启动的时间（自设备启动以来的毫秒数）
last_activity_time
此服务上次活动的时间（自启动以来，不计睡眠阶段的毫秒数）
restarting
此服务将启动的时间，如果服务当前正在运行则为 0
foreground
当服务作为前台进程运行时为 true
