电源已连接

条件 电源已连接 检查设备当前是否已连接到电源。

 

示例：
  • 仅在设备连接到交流电源适配器时打开 WiFi。
设置
电源已连接/已断开
已连接
当电源已连接时。
  • 交流电源适配器：使用交流电源适配器充电时
  • USB：通过 USB 连接充电时
  • 无线：无线充电时

已断开
当未连接电源时。

变量是一个值的容器，可在许多动作和条件中用于动态定义文本的一部分。
有关说明，请参阅动作脚本。

提供的变量
battery_level
电池的充电水平
battery_percentage
充电百分比
battery_plugged
电池是否已连接电源（1=交流电源适配器，2=USB，4=无线）或未连接（0=未连接）
battery_present
电池是否存在
battery_scale
最大电池电量
battery_status
电池状态（1=未知，2=充电中，3=放电中，4=未充电，5=已满）
battery_technology
电池技术
battery_temperature
电池温度（以十分之一摄氏度为单位）
battery_voltage
当前电池电压
battery_health
电池健康状况
battery_charge_counter (Android 5+)
电池容量，单位为微安时。不可用时为 null
battery_current_now (Android 5+)
瞬时电池电流，单位为微安（正值=充电，负值=放电）。不可用时为 null
battery_current_average (Android 5+)
平均电池电流，单位为微安（正值=充电，负值=放电）。不可用时为 null
battery_energy_counter (Android 5+)
电池剩余能量，单位为纳瓦时。不可用时为 null
