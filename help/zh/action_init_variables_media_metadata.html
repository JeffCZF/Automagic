初始化变量媒体元数据

初始化变量媒体元数据动作将媒体文件的元数据加载到局部变量中（参见下面的变量部分）。

 

示例：
  • 加载 mp3 的标题并在通知中显示标题。
设置
文件
媒体文件的路径或 URL。支持变量。
变量是存储值的容器，可在许多动作和条件中使用，以动态定义文本的一部分。
参见动作脚本说明。

提供的变量
media_cd_track_number
音频数据源在其原始录音中的顺序，或 null
media_album
专辑标题，或 null
media_artist
艺术家，或 null
media_author
作者，或 null
media_composer
作曲者，或 null
media_date
数据源创建或修改的日期，或 null
media_genre
内容类型或流派，或 null
media_title
标题，或 null
media_year
数据源创建或修改的年份，或 null
media_duration
播放持续时间（毫秒），或 null
media_num_tracks
曲目数量，或 null
media_writer
作者（如词作者），或 null
media_mimetype
MIME 类型，或 null
media_albumartist
表演者/艺术家，或 null
media_disc_number
光盘编号，或 null
media_compilation
音乐专辑合辑，或 null
media_has_audio
如果媒体包含音频内容则为 true
media_has_video (Android 4+)
如果媒体包含视频内容则为 true
media_video_width (Android 4+)
视频宽度，或 null
media_video_height (Android 4+)
视频高度，或 null
media_bitrate (Android 4+)
媒体比特率（比特/秒），或 null
media_location (Android 4.0.3+)
位置，或 null
media_video_rotation (Android 4.2+)
视频旋转角度（度），或 null
media_album_art
专辑封面，或 null
