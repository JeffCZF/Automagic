设置系统设置 (Android Wear)

动作 设置系统设置 (Android Wear) 将指定值存储到系统的设置数据库中。

警告：将系统设置修改为不支持的值可能会损害设备或导致系统崩溃。风险自负！

注意：对某些设置的更改不会立即被系统捕获，需要发生额外事件（例如关闭并重新打开屏幕）才会生效。某些设置需要通过特殊 API 更改，在设置数据库中更改此类设置的值会被系统忽略。

 

示例：
  • 更改 Automagic 中常规动作未提供的 ROM 特定设置的值。
设置
Android Wear 设备
目标设备。支持变量。
类别
  • System：用于常规设置
  • Secure：用于通常无法修改的设置
  • Global：用于对设备所有用户有效的全局设置（Android 4.2+）
名称
要修改的设置的名称。支持变量。
值
要存储在系统设置数据库中的值。支持变量。
