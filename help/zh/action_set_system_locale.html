设置系统区域

动作 设置系统区域 更改当前使用的系统区域。

警告：此功能并非 Android 官方支持，可能无法在所有设备上运行。
警告：此功能需要 root 权限，并非 Android 官方支持，可能无法在所有设备上运行。（Android 4.2+）
注意：在某些 Android 版本上，可使用 adb 授予权限，从而无需 root 即可执行此动作：

  1. 在计算机上安装 adb（搜索互联网了解如何安装 adb）
  2. 在 Android 设备的 系统设置 -> 开发者选项 中启用 USB 调试
  3. 将 Android 设备连接到计算机
  4. 在终端中执行以下命令：adb shell pm grant ch.gridvision.ppam.androidautomagic android.permission.CHANGE_CONFIGURATION

区域
要设置的区域的技术名称。支持变量。
