变量是存储值的容器，可在许多动作和条件中使用，以动态定义文本的一部分。
参见动作脚本说明。

提供的变量
device_id
设备 ID（例如 IMEI/MEID/ESN），不可用时为 null
device_software_version
软件版本（例如 IMEI/SV），不可用时为 null
line1_number
线路 1 的电话号码字符串（例如 MSISDN），不可用时为 null
network_country_iso
ISO 国家代码（仅当注册到网络时，在 CDMA 上不可靠）
network_operator
数字网络运营商名称 MCC+MNC（仅当注册到网络时，在 CDMA 上不可靠）
network_operator_name
网络运营商名称（仅当注册到网络时，在 CDMA 上不可靠）
phone_type
手机类型（0=无，1=GSM，2=CDMA，3=SIP）
sim_state
SIM 卡状态（0=未知，1=缺失，2=需要 PIN，3=需要 PUK，4=网络锁定，5=就绪）
sim_country_iso
SIM ISO 国家代码
sim_operator
SIM MCC+MNC（仅当 SIM 状态为就绪时可用）
sim_operator_name
服务提供商名称 SPN（仅当 SIM 状态为就绪时可用）
sim_serial_number
SIM 卡序列号，不可用时为 null
subscriber_id
用户 ID（例如 IMSI），不可用时为 null
voice_mail_alpha_tag
与语音信箱号码关联的字母标识
voice_mail_number
语音信箱号码，不可用时为 null
