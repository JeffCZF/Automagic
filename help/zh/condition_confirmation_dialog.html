确认对话框

条件 确认对话框 显示一个带有问题的对话框，并根据按下的按钮继续流程执行。

 

示例：
  • 打开蓝牙，但先请求确认。
设置
标题
对话框的可选标题。支持变量。
问题
要询问的问题的文本。支持变量。
False 按钮 / True 按钮
执行条件的 true/false 分支的按钮的文本标签。支持变量。
返回按钮行为
是忽略返回按钮，还是以 true 或 false 确认对话框。
激活超时
当用户在指定时间内没有响应时，是否应使用自动超时。
持续时间
自动关闭对话框并继续流程的时间量。
超时时的值
流程应继续到 true 分支还是 false 分支。
变量是一个值的容器，可在许多动作和条件中用于动态定义文本的一部分。
有关说明，请参阅动作脚本。

提供的变量
operation
用于关闭对话框的操作。

可以是以下之一：
  • yes：用户按下“是”按钮时
  • no：用户按下“否”按钮时
  • timeout：对话框因超时关闭时
  • cancel：用户按下返回键且设置了相应选项时
