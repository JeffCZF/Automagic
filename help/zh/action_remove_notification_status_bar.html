移除状态栏通知

移除状态栏通知动作移除由动作 状态栏通知 添加的指定或所有状态栏通知。

 

示例：
  • 当声音配置文件激活时显示通知，并在声音配置文件停用时移除通知。
设置
通知类型
动作应仅影响 Automagic 的通知，还是影响所有应用的通知（Android 4.3+）
Automagic 通知 ID
要移除的通知的 ID。ID 必须与动作 状态栏通知 中使用的 ID 匹配。支持变量。
全部 将移除 Automagic 的所有可移除通知，无论 ID 是什么。
通知
是移除所有可移除的通知，还是仅移除指定应用的通知。
包名
要从中移除通知的包/应用的名称。支持变量。
通知 ID
要移除的通知的 ID。支持变量。
触发器 状态栏通知已显示 会在变量 id 中提供通知的 ID。使用 {id} 来移除触发流程的通知。
过滤器
定义必须包含在某个可用字段中的文本。对于 包含文本 和 匹配通配符，支持变量。
