创建日历事件

动作 创建日历事件 在设备的指定日历上创建事件（需要 Android 4+）

 

示例：
  • 根据收到的短信文本创建日历事件。
  • 为每次收到的通话创建日历事件。
设置
日历
要创建事件的日历名称。支持变量。
开始/结束类型
事件是相对于动作执行时间创建，还是使用绝对时间创建。
开始于
事件从现在开始的偏移量。例如 1h 创建一小时后开始的事件。支持变量。
持续时间
事件的期望持续时间，如 1h 或 45m。支持变量。
从
一个脚本，其最后一条语句必须计算为日期（自1970年以来的毫秒数），用作事件的开始时间。
到
一个脚本，其最后一条语句必须计算为日期（自1970年以来的毫秒数），用作事件的结束时间。
全天
勾选后创建持续一整天的活动，事件跨越与开始于/持续时间或从/到中指定的时间段相交的所有日期。
标题
标题。支持变量。
描述
描述。支持变量。
位置
事件地点。支持变量。
提醒
要创建的提醒。支持变量。
事件可用性
可用性（并非所有日历都支持“暂定”）。
可见性
可见性。
