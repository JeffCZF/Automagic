FTP 初始化变量文件列表

FTP 初始化变量文件列表动作列出目录中与指定名称模式匹配的文件。

 

示例：
  • 获取文件列表，并仅下载小于 1MB 的文件。
设置
协议
使用常规 FTP（未加密）、FTPS（基于 SSL/TLS 的隐式 FTP，默认端口 990）还是 FTPES（基于 SSL/TLS 的显式 FTP）。
数据通道保护（仅适用于 FTPS 和 FTPES）
数据通道上传输的数据是否应加密（P）或不加密（C）。
服务器/端口
FTP 服务器的名称或 IP 及端口（默认 21）。支持变量。
用户名
连接的用户名。支持变量。
密码
连接的密码。支持变量。
变量
用于保存文件列表的变量名称。支持变量。
文件模式
要列出的文件。文件模式可以包含通配符，如 * 和 ?。支持变量。
包含文件属性
选中时，为每个文件返回包含路径、文件类型、文件大小和文件日期的属性列表。
变量是存储值的容器，可在许多动作和条件中使用，以动态定义文本的一部分。
参见动作脚本说明。

提供的变量
files
文件列表
