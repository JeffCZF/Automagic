媒体会话已更改（实验性）

触发器 媒体会话已更改（实验性） 在指定播放器的媒体会话发生更改（播放新曲目、加载媒体图像等）时执行。仅位置发生常规更改不会调用触发器。

 

示例：
  • 在叠加小部件中显示当前曲目的标题。
设置
包名称
要监视的媒体播放器的名称。支持 glob 模式（*, ?）。
变量是一个值的容器，可在许多动作和条件中用于动态定义文本的一部分。
有关说明，请参阅动作脚本。

提供的变量
triggertime
触发器执行的时间
playback_state
当前状态（0=无，1=已停止，2=已暂停，3=正在播放，4=快进，5=快退，6=缓冲，7=错误，8=连接中，9=跳到上一个，10=跳到下一个，11=跳到队列中的项目）
playback_position
当前位置（毫秒）
title
标题
subtitle
副标题
description
描述
icon
当前图像
icon_uri
加载图像的 URI
media_duration
曲目的持续时间（毫秒）
